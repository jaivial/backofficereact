Usage Instructions

Basic tooltip: Add data-tooltip="Your message" to any element.

Position: Use keywords in the attribute: data-tooltip="Message|top" or data-tooltip-position="bottom".

Light theme: Add data-tooltip-theme="light".

No arrow: Add data-tooltip-no-arrow attribute.

Accessibility: Always include aria-describedby="unique-id" (or let the tooltip text serve as description).

Disabled buttons: Wrap in a container div for tooltip support.
